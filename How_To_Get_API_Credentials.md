# 🔑 How to Get Telegram API Credentials

To use the **Telegram ExactTime Scheduler**, you need an `API ID` and `API Hash` from Telegram. Follow these steps:

---

## Step-by-Step Guide

### 1. Visit the Website
Go to **[https://my.telegram.org/auth](https://my.telegram.org/auth)** in your browser.

### 2. Login
Enter your Telegram phone number in **international format** (e.g., `+8801700000000` for Bangladesh) and click **Next**.

### 3. Enter OTP Code
You will receive a login code inside your **Telegram app** (from the official *Telegram* service account).

> ⚠️ **Never share this code with anyone!**

Enter the code on the website to log in.

### 4. Go to API Tools
Once logged in, click on **"API development tools"**.

### 5. Create an App
If this is your first time, you'll see a form. Fill it in like this:

| Field | What to Enter |
|-------|--------------|
| **App title** | Any name (e.g., `MyScheduler`) |
| **Short name** | 5–32 alphanumeric chars (e.g., `msched01`) |
| **URL** | Leave blank |
| **Platform** | `Desktop` or `Other` |
| **Description** | Leave blank or write anything |

Click **Create application**.

### 6. Copy Your Credentials
You'll see your credentials on the next screen:

- **API ID** — a number like `1234567`
- **API Hash** — a long string like `abcd1234ef5678...`

Copy both and paste them into the **Telegram ExactTime Scheduler** app.

---

> 💡 **Tip:** Your session is saved after first login, so you won't need to re-enter OTP every time.
